Developer information
=====================

This chapter contains information for developers, who want to contribute to *Blender.LectureEdit*.

Currently, this documentation is somewhat lacking and the source code does not contain many helpful comments either.
If you wish to have a feature or a bugfix, it's probably best to contact the developers via `Github <https://github.com/ls1intum/Blender.LectureEdit>`_.
